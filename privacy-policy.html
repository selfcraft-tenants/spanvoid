Privacy Policy

Last updated: April 2, 2026

This Privacy Policy describes how Spanvoid ("we", "us", or "our") handles information when you interact with our services and content across social media platforms including Instagram, Pinterest, Threads, YouTube, and TikTok.

1. Information We Collect

We collect minimal information necessary to operate our content publishing services:

  • Account Information: When connecting third-party platforms (such as Pinterest, TikTok, YouTube) via OAuth, we receive an access token and basic profile information (username, account ID) as authorized by you.
  • Usage Data: We may collect aggregated, non-personally-identifiable analytics data (impressions, saves, clicks) provided by platform APIs to improve our content.

2. How We Use Information

We use collected information solely to:

  • Publish and manage content on connected platforms on behalf of the account owner
  • Analyze content performance to improve quality and relevance
  • Maintain and secure platform connections

3. Pinterest API Data

When using Pinterest API services:

  • We access only the data scopes explicitly authorized during the OAuth flow (boards, pins)
  • We do not store Pinterest API data beyond what is necessary for immediate content operations
  • We do not sell, share, or transfer Pinterest user data to third parties
  • All API interactions comply with Pinterest Developer Guidelines
  • Access tokens are stored securely and refreshed as required by Pinterest's token lifecycle

4. TikTok API Data

When using TikTok API services:

  • We access only the data scopes explicitly authorized during the OAuth flow (video.publish, user.info.basic)
  • We use the Content Posting API to publish video content on behalf of the authorized account owner
  • We do not store TikTok API data beyond what is necessary for immediate content operations
  • We do not sell, share, or transfer TikTok user data to third parties
  • All API interactions comply with TikTok API Terms of Service
  • Access tokens are stored securely and refreshed as required by TikTok's token lifecycle

5. YouTube API Data

When using YouTube API services:

  • We access only the data scopes explicitly authorized during the OAuth flow
  • We do not store YouTube API data beyond what is necessary for immediate content operations
  • All API interactions comply with YouTube API Terms of Service

6. Data Storage and Security

We take reasonable measures to protect information:

  • Access tokens and credentials are stored in encrypted environment variables, never in public repositories
  • We use HTTPS for all API communications
  • Access is restricted to authorized account owners only

7. Data Sharing

We do not sell, rent, or share personal information with third parties. Data is only shared with platform APIs (Instagram, Pinterest, Threads, YouTube, TikTok) as necessary to publish content you have authorized.

8. Data Retention

We retain platform connection data (access tokens, account IDs) only for as long as the service connection is active. Performance analytics are retained in aggregated form. You may request deletion of your data at any time.

9. Your Rights

You have the right to:

  • Revoke platform access at any time through the respective platform's settings
  • Request information about what data we hold
  • Request deletion of your data

10. Third-Party Services

Our services interact with third-party platforms governed by their own privacy policies:

  • Instagram Privacy Policy
  • Pinterest Privacy Policy
  • Threads Privacy Policy
  • TikTok Privacy Policy
  • YouTube / Google Privacy Policy

11. Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be reflected by updating the "Last updated" date above.

12. Contact

For questions about this Privacy Policy or to exercise your data rights, please contact us through our social media channels or open an issue in our repository.
